Drop your YouTube channel profile photo here as:

    profile.jpg

It will be displayed:
  - Top-left of every page, clipped to a circle with a cyan glow ring
  - In the About section (Section 7), larger circular crop

If this file is missing, the site falls back to showing "NN" initials
automatically — so you won't see a broken image icon.

Recommended:
  - Square image (at least 400x400)
  - JPG or PNG
  - File size under 200 KB for fast loading
